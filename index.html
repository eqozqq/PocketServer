If you still want other people to join your server wich are on a different network, you must setup portforwarding. This step cannot be explained here since for every network, the way of setting this up will be different.

You can find a well explained guide on the following link: https://portforward.com

On a IPv4 network, there are is a local and public IP.

The local IP is ment to identify you device in a local network, it cannot accesed on the internet. Use the local IP to join your server.

The public IP can be accessed over the internet. Use it to invite other people outside your network. (Portforwarding required first)

You do not need to use these IP's to join your server. The server will be automatically shown on the friends tab in the game.

These IP's are ment to be used for development or testing.

Starting from Android 11, apps can no longer access /sdcard location and only can modify files in their own directories. You can however import and export your files within the app easily.

  • Select the files that you want to export
  • If you have multiple files and do not want to export each file manually, zip selected files
  • Long press on the created zip file and export to external storage
